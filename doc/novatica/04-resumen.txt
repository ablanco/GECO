resumen:

GECO pretende ser un gestor de contraseñas distribuido con diferentes
interfaces de comunicación con el usuario. Presentanto tanto una
interfaz de escritorio en GTK como una interfaz web desarrollada con
python y web.py.

keywords: contraseñas, seguridad, python, gtk+, web.py, software libre

historial-academico:
